# Harpoon
VSCode version of [harpoon](https://github.com/ThePrimeagen/harpoon) made by [ThePrimeagen](https://www.youtube.com/@ThePrimeagen).

# Setup
``` JSON
{
    "key": "ctrl+e",
    "command": "harpoon.show"
},
{
    "key": "ctrl+h",
    "command": "harpoon.yoink",
    "args": [0]
},
{
    "key": "ctrl+t",
    "command": "harpoon.yoink",
    "args": [1]
},
{
    "key": "ctrl+n",
    "command": "harpoon.yoink",
    "args": [2]
}
```
